# Bloc-Note Web

## Auteur

Grah Lowa Vincent Ezechiel

## Description du projet

Ce projet est un **bloc-note  développé avec **Visual Studio Code**. Il permet à l’utilisateur de **créer, éditer et enregistrer des notes** directement dans le navigateur.

L’objectif est de proposer une interface simple et intuitive pour prendre des notes rapidement et les organiser.

## Objectifs

* Permettre la création et l’édition de notes
* Sauvegarder et récupérer les notes de manière simple
* Offrir une interface épurée et facile à utiliser

## Fonctionnalités principales

* Créer une nouvelle note
* Éditer une note existante
* Supprimer une note
* Sauvegarder les notes localement (dans le navigateur ou via stockage local selon l’implémentation)

## Technologies utilisées

*C#
* Visual Studio Code (environnement de développement)

## Contexte

Ce projet a été réalisé comme exercice pratique pour renforcer mes compétences  avec un accent sur l’interactivité et la gestion des données côté client.
